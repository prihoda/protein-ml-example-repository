# Protein ML - Example repository

## Local setup

Install dependencies into a new conda environment:

```
conda create -y -n protein-ml-exploration python=3.11
conda activate protein-ml-exploration
pip install -r requirements.txt
```

Run Jupyter Lab:

```
jupyter lab
```
